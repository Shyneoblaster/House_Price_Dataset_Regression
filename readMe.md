# Predicción de Precios de Viviendas (House Pricing Regression)

Este proyecto aplica técnicas de aprendizaje supervisado para predecir el valor de mercado de viviendas utilizando el **House Prices Dataset** de Kaggle. Se comparan modelos de ensamble y redes neuronales para evaluar el desempeño en tareas de regresión.

## Requisitos
* Python 3.10+
* Pandas & NumPy
* Matplotlib & Seaborn
* Scikit-Learn
* TensorFlow/Keras o Scikit-Learn
